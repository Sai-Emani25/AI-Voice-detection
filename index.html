About the Project

This is an API-based system designed to detect whether a voice sample is AI-generated or Human-generated. It supports multiple languages (Tamil, English, Hindi, Malayalam, Telugu, Kannada) and accepts Base64-encoded MP3 audio inputs.

Built with FastAPI, the system provides a RESTful API with real-time voice detection capabilities using WebSocket, and features a modern web interface for easy interaction.

API Specification

POST /detect

Request Body:

{ "audio_base64": "SUQzBAAAAA...", "language": "English" }

Response:

{ "classification": "AI-Generated", "confidence_score": 0.95, "explanation": "High confidence based on spectral artifacts...", "metadata": { ... } }
